Atelier

Git et Github pour débutants

Description

Versionner son code n'est pas un détail technique réservé aux professionnels : c'est une compétence de base que tout développeur doit maîtriser dès le début. Cet atelier pratique vous apprend à utiliser Git et GitHub pas à pas, à travers des exercices concrets — créer un dépôt, faire des commits, gérer des branches, résoudre un conflit — pour ne plus jamais avoir à renommer un fichier "projet_final_v2_definitif".

Programmes

  • 10h00 — Introduction : pourquoi versionner son code
  • 10h20 — Installation et configuration de Git
  • 10h45 — Premiers commits, comprendre le staging
  • 11h30 — Branches et fusion (merge)
  • 12h15 — Mini-exercice en binôme : résoudre un conflit
  • 12h45 — Questions et clôture
